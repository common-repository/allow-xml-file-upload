== Description ==

A super simple solution for Import any XML files to WordPress with Media Uploader



== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/allow-xml-file-upload` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
